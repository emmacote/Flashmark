-- Learning Machine --
Learning Machine is a web application built to act as a learning aid.
It'set up with the following features....
<br />
<ul>
    <li>Make and tag learning topics</li>
    <li>Create and practice exercises</li>
    <li>Make up a list of support resources for your topics of interest</li>
    <li>Review history of attempts at practice exercises to see where improvement is needed.</li>
</ul>
